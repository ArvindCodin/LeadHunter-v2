Built-in Backend (Active on Replit)

This app is running on Replit with a built-in Node.js + Playwright backend already connected. The backend connector in the sidebar is pre-filled with your Replit URL. Just click TEST to verify the connection, then start hunting leads.

If you want to use an external backend instead, paste its URL in the left sidebar connector and click TEST.

How it works

The frontend sends requests like:

GET /api/scrape?q=dental+clinic+in+Anna+Nagar+Chennai&max=20 Response: { "results": [...real businesses...], "count": 14 }

The Playwright scraper runs headless Chromium on the server, bypassing Google Maps CORS restrictions. When no backend is connected, the app uses realistic simulation.
