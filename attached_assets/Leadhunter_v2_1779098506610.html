Why you need a backend

Browsers block direct Google Maps scraping (CORS policy). Your Playwright scraper needs to run on a server that the HTML can call. Here's the easiest free setup using Replit — no credit card, takes 5 minutes.

Step-by-step: Deploy on Replit (Free)

  1. Go to replit.com → Create account → New Repl → choose Python
  2. Paste this server code into main.py (copy below)
  3. In Replit Shell, run: pip install flask playwright flask-cors && playwright install chromium
  4. Click Run → Replit gives you a URL like https://yourname.repl.co
  5. Paste that URL in the backend box on the left sidebar → click TEST

main.py — Paste this into Replit

from flask import Flask, request, jsonify from flask_cors import CORS from playwright.sync_api import sync_playwright import re, time, random app = Flask(__name__) CORS(app) # allows your HTML to call this CHAIN_KEYWORDS = [ 'naturals','ccd','cafe coffee day','saravana stores','saravana bhavan', 'kfc','mcdonalds','dominos','pizza hut','subway','burger king', 'apollo pharmacy','medplus','wellness forever','reliance','spencer', 'lifestyle','shoppers stop','decathlon','zara','h&m','tanishq', 'muthoot','manappuram','tnstc','bescom','bsnl','airtel store', 'jio point','samsung','apple store','samsung','lg showroom' ] def is_chain(name): n = name.lower() return any(k in n for k in CHAIN_KEYWORDS) def scrape_maps(query, max_results=20): results = [] with sync_playwright() as p: browser = p.chromium.launch(headless=True) page = browser.new_page() page.goto(f"https://www.google.com/maps/search/{query.replace(' ','+')}", timeout=20000) page.wait_for_timeout(3000) # Scroll to load results for _ in range(4): page.keyboard.press('End') page.wait_for_timeout(1000) listings = page.query_selector_all('[role="article"]') for item in listings[:max_results]: try: name_el = item.query_selector('[class*="fontHeadlineSmall"]') name = name_el.inner_text() if name_el else '' if not name or is_chain(name): continue rating_el = item.query_selector('[class*="fontBodyMedium"] span[aria-label]') rating_text = rating_el.get_attribute('aria-label') if rating_el else '' rating_match = re.search(r'(\d+\.?\d*)', rating_text) rating = float(rating_match.group(1)) if rating_match else 0.0 review_match = re.search(r'([\d,]+)\s+review', rating_text) reviews = int(review_match.group(1).replace(',','')) if review_match else 0 # Check for website button has_website = bool(item.query_selector('a[data-item-id="authority"]') or item.query_selector('[aria-label*="website" i]')) category_el = item.query_selector('[class*="fontBodyMedium"]:not([aria-label])') category = category_el.inner_text().strip() if category_el else query.split(' in ')[0] results.append({ 'name': name, 'category': category, 'area': query.split(' in ')[-1].replace(' Chennai',''), 'rating': rating, 'reviews': reviews, 'hasWebsite': has_website, 'source': 'playwright' }) except: continue browser.close() return results @app.route('/scrape') def scrape(): query = request.args.get('q','') max_r = int(request.args.get('max', 20)) if not query: return jsonify({'error': 'no query'}), 400 try: data = scrape_maps(query, max_r) return jsonify({'results': data, 'count': len(data)}) except Exception as e: return jsonify({'error': str(e)}), 500 @app.route('/ping') def ping(): return jsonify({'status': 'ok', 'message': 'LeadHunter backend running'}) if __name__ == '__main__': app.run(host='0.0.0.0', port=8080)

Alternative: Railway.app (also free)

If Replit is slow, use Railway — slightly more steps but faster servers:

  1. Go to railway.app → New Project → Deploy from GitHub
  2. Create a repo with the code above + a requirements.txt:
flask flask-cors playwright

Railway auto-detects Python, gives you a URL in ~2 minutes. Free $5/month credit = plenty for scraping.

How it connects

Once running, paste your URL in the left sidebar. LeadHunter will send requests like:

GET https://your-url.repl.co/scrape?q=dental+clinic+in+Anna+Nagar+Chennai&max=20 Response: { "results": [...real businesses...], "count": 14 }

When no backend is connected, the app uses realistic simulation so you can still test the scoring engine and UI.
